## Installation

You can install **bdXdata** from [our
repository](https://crp2a.r-universe.dev) with:

``` r
options(repos = c(crp2a = 'https://crp2a.r-universe.dev',
                  CRAN = 'https://cloud.r-project.org'))
install.packages("bdXdata")
```
